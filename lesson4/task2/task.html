We can also test the contents of strings using these functions.

string.isalnum() # test if string contains only letters and numbers
string.isalpha() # test if string contains only letters
string.isdigit() # test if string is all digits
string.istitle() # test if string contains only title words (upper case for the first letter of any words, lower case for the rest)
string.isupper() # test if string is all upper case
string.islower() # test if string is all lower case
string.isspace() # test if string contains only spaces

So if string = "HELLO"
string.isalnum() = True
string.isdigit() = False
string.isupper() = True
